ИНСТРУКЦИЯ ПО ЗАПУСКУ (VR ВЕРСИЯ):

1. УСТАНОВКА ДРАЙВЕРА VMT (Обязательно для SteamVR):
   - Скачайте VMT: https://github.com/gpsnmeajp/VirtualMotionTracker/releases
   - Распакуйте и запустите "VMT_Manager.exe".
   - Нажмите "Install" и перезапустите SteamVR.
   - В настройках SteamVR (Устройства -> Контроллеры -> Настройка управления) 
     должны появиться виртуальные трекеры/контроллеры.

2. УСТАНОВКА БИБЛИОТЕК:
   pip install -r requirements.txt

3. ЗАПУСК:
   - Сначала запустите SteamVR.
   - Затем запустите скрипт: python tracking.py
   - Выберите камеру.

4. УПРАВЛЕНИЕ:
   - Движение руки: Ориентируется по запястью.
   - Глубина (Z): Рассчитывается автоматически (чем ближе рука к камере, тем она ближе в VR).
   - Триггер (Нажатие): Соедините кончики большого и указательного пальцев ("щипок").
   - Выход: 'q' в окне видео.
